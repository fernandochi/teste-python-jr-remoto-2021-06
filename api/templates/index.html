Instruct - Magpy

Uma api para registrar projetos e seus módulos de desenvolvimento, deste modo outros desenvolvedores podem ter acesso ao projeto e seus requisitos. Sendo possível que seja determinada explicitamente qual versão cada módulo está utilizando ou, caso seja a útlima versão, o própri sistema identifica qual seja

get /api/projects
Busca projetos (buscaProjetos)
Retorna todos os projetos disponíveis ou, se for passado um nome como parâmetro traz o projeto com este nome

Query parameters

nomeProjeto (optional)
Query Parameter — Parâmetro opcional do nome do projeto

Return type

array[ProjectList]

Example data

Content-Type: application/json 
{}

Produces

This API call produces the following media types according to the Accept request header; the media type will be conveyed by the Content-Type response header.
  • application/json

Responses

200

Retorna o projeto(s)

404

Projeto com o nome especificado não existe
Up 
post /api/projects
Cria um projeto (criaProjeto)
Cria um novo projeto com o nome passado e os pacotes específicados, caso a versão não for explicíta irá utilizar a última versão daquele pacote

Consumes

This API call consumes the following media types via the Content-Type request header:
  • application/json

Request body

project ProjectInput (optional)
Body Parameter —

Return type

ProjectList

Example data

Content-Type: application/json 
{"empty": false}

Produces

This API call produces the following media types according to the Accept request header; the media type will be conveyed by the Content-Type response header.
  • application/json

Responses

201

Quando todos os pacotes estão corretos ProjectList

400

Se um pacote inexistente ou a sua versão não existir, um erro é retornado de bad_request Error

delete /api/projects
Deleta um projeto (deleteProject)
Deleta um projeto com o nome passado no parâmetro ou retorna 404 se o projeto não existir

Produces

This API call produces the following media types according to the Accept request header; the media type will be conveyed by the Content-Type response header.
  • applicaton/json

Responses

204

O projeto foi deletado

404

Não existe projeto com o nome passado
